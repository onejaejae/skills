# Presentation Research Reference

## Structure Frameworks

| Framework | Best For | Core |
|-----------|----------|------|
| 기승전결 | 컨퍼런스, 인사이트 공유 | 반전(轉)이 핵심 |
| 3막 구조 | 범용 스토리텔링 | Setup → Confrontation → Resolution |
| Pyramid (Minto) | 경영진 보고 | 결론 먼저 (BLUF) + SCQA |
| PSB | 설득/영업 | Problem → Solution → Benefit |
| STAR | 사례 발표 | Action이 60% |
| Sparkline (Duarte) | 비전/동기부여 | What Is ↔ What Could Be 교대 |

## Storytelling Techniques

- **Hook (30초)**: 충격 통계, 개인 스토리, 도발적 질문, What-if, 반직관적 주장
- **Callback**: 오프닝 요소를 클로징에서 재활용 → 완결성
- **Tension-Release**: 문제 펼치기(긴장) → 해결 공개(해소)
- **Rule of Three**: 핵심 포인트 최대 3개, 3번 반복

## Transition Types (MIT CommLab)

| Type | Example |
|------|---------|
| 언어적 | "지금까지 X를 봤습니다. 이제 Y로..." |
| 음성적 | 새 섹션 전 속도 낮추기 |
| 시각적 | 섹션 구분 슬라이드, 빈 슬라이드 |
| 침묵 | 중요 포인트 후 3-5초 pause |

## Coaching: 4P Voice Framework

- **Power**: 중요 순간 볼륨 UP, 때로 낮춰서 집중 유도
- **Pitch**: 문장 끝 음 내려야 확신있어 보임
- **Pace**: 일반 150-160 WPM, 강조 90-110 WPM, 전환 180-200 WPM
- **Pause**: 핵심 포인트 후 2-3초 → 정보 흡수율 40% 향상

## Minimal Presentation (Garr Reynolds)

- 슬라이드 1장 = 아이디어 1개
- 초안의 30% 삭제하면 강해짐
- PRUNE: Platitudes, Redundancy, Unnecessary words, Needless complication, Excursions 제거
- 여백은 강조 도구

## Time Distribution (60min)

| Section | % | Time |
|---------|---|------|
| Hook + Opening | 10% | 6min |
| Body (3 points) | 70% | 42min |
| Closing + CTA | 15% | 9min |
| Buffer | 5% | 3min |
