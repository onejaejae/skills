# Research Output

Use this reference when turning browser output into a user-facing answer.

## Report shape

- Start with access status
- Then list findings
- Then include evidence
- Then separate inference from observation

## Evidence rules

- Cite the exact URL when possible
- Include the page title
- Mention whether the page came from an authenticated browser session
- If a claim came from the page text, quote only the minimum needed to identify it

## Good behavior

- Say when a page could not be reached
- Say when a page was only partially rendered
- Say when a result is based on inference
- Ask for a screenshot or pasted text only after browser-based extraction has genuinely failed
