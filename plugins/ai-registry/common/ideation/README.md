# Ideation Plugin

모호한 아이디어를 구조화된 발산-수렴 프로세스를 통해 구체적인 계획으로 전환하는 플러그인.

## Features

- **5단계 구조화 프로세스**: Seed Capture → Discover → Define → Ideate → Design
- **발산-수렴 사이클**: 문제 탐색(발산) → 문제 정의(수렴) → 솔루션 생성(발산) → 구체화(수렴)
- **복잡도별 스케일링**: Light / Medium / Deep 3단계로 자동 조절
- **Anti-pattern 방지**: 조기 수렴, 앵커링, 얕은 다양성 등 방지

## Skills

### ideation

아이디어 탐색 및 구체화 스킬.

**트리거:**
- `ideation`, `아이디에이션`
- `brainstorm`, `브레인스토밍`
- `아이디어`, `explore ideas`
- `어떻게 하면 좋을까`, `뭐가 좋을까`
- `도입하고 싶어`

## Installation

```bash
claude plugins:install ai-registry/ideation
```

## Usage

```bash
/ideation 캐시 시스템 도입하고 싶어
/ideation 마이크로서비스 전환 검토
```
