# explaining-code

코드 동작 원리 설명 스킬 - 구체적 데이터 트레이스와 구조화된 설명으로 코드 이해를 돕습니다.

## 설치

```bash
/plugin install explaining-code@ai-registry
```

## 사용법

### 트리거

- "이 코드 설명해줘", "이게 어떻게 동작해?"
- "왜 이렇게 작성했어?", "이 설계 의도가 뭐야?"
- 온보딩, 멘토링, 코드 리뷰 맥락 설명

### 핵심 원칙

1. **구체적 데이터로 추상을 깨라** - 실제 값으로 트레이스
2. **전체에서 부분으로 내려가라** - 조감도 먼저, 디테일은 나중에
3. **"왜 이 방식인가"를 답하라** - 코드가 아닌 의도를 설명

### 설명 깊이

| 복잡도 | 깊이 | 기준 |
|---|---|---|
| 파일 1개 이하 | Quick | 한 문장 요약 + 트레이스 + 주의점 |
| 파일 2-3개 | Standard | + 비유 + 다이어그램 + Why-Not |
| 파일 4개+ | Deep Dive | + 시퀀스 + Before/After + 설계 결정 |

### 설명 기법

- 한 문장 요약
- 비유 (Analogy)
- 다이어그램 (흐름도, 구조도, 시퀀스, 상태도)
- Phase별 책임 테이블
- 예시 데이터 트레이스
- Before/After 상태 비교
- Why-Not 설명 (Counterfactual)
