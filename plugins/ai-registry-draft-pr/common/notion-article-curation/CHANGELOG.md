# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-13

### Added
- 팀 공유 Notion DB (솔루션개발팀 위키) 연동
- 중복 체크, 콘텐츠 분석, 자동 분류 워크플로우
- 대량 처리 (30개+) 병렬 Agent 지원

### Changed
- 개인 스킬에서 팀 공유 플러그인으로 전환
- Data Source ID를 팀 공유 DB로 변경 (`314d87e5-17f4-8198-be4c-000b127b0496`)

### Removed
- `Read` 체크박스 property (팀 DB에 없음)
